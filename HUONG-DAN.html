2 Cài đặt & mở lần đầu

App chưa ký chứng chỉ trả phí (bình thường với app tự build) nên lần đầu hệ điều hành sẽ cảnh báo. Cách qua:

🍎 macOS

Mở file .dmg → kéo app vào Applications. Nếu bị chặn: chuột phải vào app → Open → Open (chỉ làm 1 lần đầu).

🪟 Windows

Bấm chạy file .exe để cài. Nếu hiện "Windows protected your PC": bấm More info → Run anyway. Lần đầu mở app, nếu Firewall hỏi → tick cả Private và Public rồi Allow.

3 Dùng app

  1. Bên trái chọn scene (Floor Spots / Bomb Timer).
  2. Bấm Start NDI ở scene đó (đèn xanh = đang phát), hoặc Start All phát cả 2.
  3. Chọn FPS (30 mặc định / 60). Chỉnh thông số ở panel dưới.
  4. Điều khiển bằng phím tắt — chỉ scene đang xem mới nhận phím.

⌨️ Phím tắt

ScenePhím
Floor Spots →/Space bật spot tiếp · ← lùi · A bật hết · R reset · 1–9 nhảy tới spot
Bomb Timer Space start/pause · R reset · O reveal "OPEN IT"

4 Nhận trong TouchDesigner

  1. Tạo NDI In TOP → ô Source Name chọn … (IllogicalFloor) → map ra sàn.
  2. Tạo NDI In TOP thứ 2 → chọn … (IllogicalTimer) → map ra tường.
Nếu ô Source Name TRỐNG (không thấy source): làm theo thứ tự bên dưới — app phát đúng, chỉ là TD chưa "thấy".
  1. Bấm vào ô Source Name — nó là menu xổ xuống, không phải gõ sẵn. Bấm để xem danh sách rồi chọn.
  2. Menu trống → gạt Active Off → On để quét lại.
  3. Vẫn trống → điền Extra Search IPs rồi Active Off→On:
    • TD cùng máy với app → 127.0.0.1
    • TD máy khác cùng mạng → IP LAN của máy chạy app
  4. Vẫn trống → bấm Source Name và gõ tay tên source, ví dụ:
    MACBOOK-PRO-CUA-MACOS-2.LOCAL (IllogicalFloor)
  5. Còn nữa → kiểm tra Firewall (mục dưới).

Tìm IP LAN của máy chạy app

HệCách lấy IP

🪟 WindowsCMD: ipconfig → dòng IPv4 Address của card đang dùng
⚠️ IP LAN đổi khi reconnect Wi-Fi/router. Nếu mai mốt mất source, kiểm tra lại IP. Muốn cố định: đặt DHCP reservation trên router hoặc set IP tĩnh.

6 Lưu ý chạy sự kiện

★ Chạy sự kiện trên máy Mac (app + TD cùng 1 máy)

  • Trong NDI In TOP dùng Extra Search IPs = 127.0.0.1 → ổn định nhất, không lo IP đổi khi đổi mạng ở venue.
  • Mỗi NDI In TOP set Output đúng res: sàn 3305×2561, tường 8324×815.
  • Nền đen tuyệt đối → comp Add/Screen nếu cần (Bomb Timer).
  • Phát 2 stream lớn khá nặng → giữ FPS 30, chỉ Start NDI scene đang dùng. Nếu giật, tắt bớt stream không cần.
  • Chỉ scene đang active nhận phím → trước khi bấm phím, click chọn đúng scene ở sidebar.

🔥 Firewall (khi chạy 2 máy qua LAN)

macOS: System Settings → Network → Firewall → tắt tạm, hoặc Options… cho phép Illogical Control + TouchDesigner nhận kết nối.

Windows: lần đầu mở app tick cả Private + Public. Lỡ rồi: Control Panel → Windows Defender Firewall → Allow an app → thêm app + TD.
